# 투자 콘텐츠 인텔리전스 OS 시각화 카드 디자인 아이디어

작성일: 2026-05-05  
대상: Investment Content Dashboard / Investment Content Intelligence OS  
목적: 패널·채널·종목별 공개 투자 콘텐츠 발언의 성과를 모바일에서 빠르게 이해하고, 상세 검증과 소셜 공유까지 이어지는 카드 디자인 체계를 정의한다.

---

## 1. 디자인 원칙

### 1.1 서비스 포지션

이 서비스는 AI 투자 추천 서비스가 아니라 공개 투자 콘텐츠 발언을 기록·검증·리그화·차트화하는 콘텐츠 인텔리전스 OS다.

따라서 모든 카드에는 다음 맥락이 유지되어야 한다.

- 투자 권유 아님
- 공개 콘텐츠 발언 기준
- 추천일 종가 대비
- 기준일 명시
- 근거 발언 또는 영상/문서 출처 연결
- GOOD / FLAT / MISS는 비난이 아니라 성과 분류값으로 표현

### 1.2 시각 스타일

현재 제품의 토스형 라이트 디자인 시스템을 유지한다.

- 밝은 배경: `#f7f9fc`
- 흰 카드: `#ffffff`
- 텍스트: `#191f28`
- 보조 텍스트: `#6b7684`
- 포인트 블루: `#3182f6`
- 얇은 회색 보더: `#e5e8eb`
- 큰 radius와 약한 shadow
- 모바일 우선 정보 밀도
- 한글/영문 혼합 텍스트는 과한 letter-spacing 금지

### 1.3 카드 체계

카드는 하나의 디자인으로 통일하지 않고 역할별로 나눈다.

```text
요약 카드 → 비교 카드 → 근거 카드 → 공유 카드 → 리그 카드
```

추천 사용 흐름:

```text
메인에서는 대안 1로 시선을 잡고,
대안 4로 빠르게 스캔시키고,
상세에서는 대안 3으로 검증 신뢰를 만들고,
바이럴에서는 주간 리포트/성적표/누가 먼저 말했나 카드로 공유를 유도한다.
```

## 2. 기존 4가지 모바일 카드 대안

## 2.1 수익률 헤드라인 카드

기존 대안 1 기반.

### 용도

- 메인 상단 대표 성과
- 패널 상세 첫 카드
- 주간 리포트 하이라이트
- 공유용 썸네일

### 역할

“이 발언 이후 결과가 어땠는지”를 가장 빠르게 보여준다.

```text
┌─────────────────────────────┐
│ 신사임당 샘플        GOOD    │
│                             │
│   +6.84%        ▲ 78원       │
│   추천일 종가 대비            │
│                             │
│   1,140 ───────▶ 1,218       │
│   추천일          현재        │
│                             │
│ 부자들이 현금 비중을 줄이기… │
│ 04.27 추천 · 05.05 기준       │
└─────────────────────────────┘
```

### 디자인 방향

- 수익률을 가장 크게 보여준다.
- `추천일 종가 대비` 문구를 반드시 넣는다.
- 상승/하락 색상은 초록/파랑/회색 등으로 중립적으로 사용한다.
- `GOOD`, `FLAT`, `MISS`는 작은 배지로 표현한다.

### 추천 위치

- 메인 페이지 상단
- 패널 상세 상단
- 주간 리포트 대표 카드
- 단일 성과 공유 이미지

---

## 2.2 리스트 친화형 카드

기존 대안 4 기반.

### 용도

- 여러 패널 퍼포먼스 목록
- 종목별 발언 타임라인
- 채널별 최근 발언 리스트
- 모바일 스크롤 피드

### 역할

많은 항목을 빠르게 훑어보게 한다.

```text
┌─────────────────────────────┐
│ 신사임당 샘플       +6.84% ▲ │
│ ─────────────────────────── │
│ 1,140 → 1,218원    (+78원)   │
│                             │
│ 부자들이 현금 비중을…        │
│ 04.27 추천 · 05.05 기준       │
└─────────────────────────────┘
```

### 디자인 방향

- 카드 높이를 낮춘다.
- 우측 상단에 성과를 배치한다.
- 가운데에 가격 변화를 표시한다.
- 하단에 콘텐츠 제목과 날짜를 표시한다.
- 모바일에서는 이 카드가 기본 리스트 카드가 된다.

### 추천 위치

- 메인 하단 리스트
- 패널별 최근 발언
- 종목별 추천 타임라인
- 주간 리포트 항목 리스트

---

## 2.3 좌우 대비 카드

기존 대안 3 기반.

### 용도

- 추천일 vs 현재 비교
- 복기/교육 콘텐츠
- 상세 분석
- 블로그/링크드인 이미지용

### 역할

“그때 가격과 지금 가격”을 가장 명확하게 비교한다.

```text
┌─────────────────────────────┐
│ 신사임당 샘플 · 04.27        │
├──────────────┬──────────────┤
│ 추천일        │ 현재          │
│              │              │
│ 1,140원       │ 1,218원 ▲     │
├──────────────┴──────────────┤
│        +6.84%  (+78원)       │
│        추천일 종가 대비       │
├─────────────────────────────┤
│ 부자들이 현금 비중을 줄이기… │
└─────────────────────────────┘
```

### 디자인 방향

- 교육/복기용이므로 정보 설명이 조금 더 많아도 된다.
- `추천일 종가 대비`를 반드시 포함한다.
- 상승률만 보이지 않도록 `추천일`, `현재`, `기준일`을 명확히 표시한다.

### 추천 위치

- 패널 상세
- 종목 상세
- 공유 리포트
- 블로그/뉴스레터용 이미지

---

## 2.4 스파크라인 카드

기존 대안 2 기반.

### 용도

- 종목 상세
- 차트 overlay 축약판
- 패널 발언 이후 가격 흐름
- 프리미엄/상세 기능처럼 보이게 하는 UI

### 역할

숫자뿐 아니라 흐름을 보여준다.

```text
┌─────────────────────────────┐
│ 신사임당 샘플       +6.84%   │
│             ╱╲              │
│        ___╱  ╲___╱╲         │
│                             │
│ 1,140원 ───▶ 1,218원         │
│ 추천일 종가 대비 +78원       │
│                             │
│ 부자들이 현금 비중을…        │
└─────────────────────────────┘
```

### 디자인 방향

- 실제 차트 데이터가 붙기 전에는 장식 차트처럼 보이지 않게 주의한다.
- 실제 가격 데이터가 연결된 후 사용하는 것이 좋다.
- 스파크라인 위에 추천일 marker를 찍으면 효과가 강하다.

### 추천 위치

- 종목 상세
- 패널 상세 고급 카드
- 차트 overlay preview
- 향후 고급 기능 영역

## 3. 추가 시각화 카드 아이디어

## 3.1 패널 성적표 카드

### 용도

패널 한 명의 최근 성과를 요약한다.

```text
┌─────────────────────────────┐
│ 패널 성적표                  │
│ 신사임당 샘플                │
│                             │
│ GOOD 4   FLAT 2   MISS 1    │
│ ▰▰▰▰▱▱▱  최근 3개월          │
│                             │
│ 평균 추천일 대비   +4.2%     │
│ 표본 수             7건       │
│ 리그               Major     │
│                             │
│ 공개 콘텐츠 발언 기준         │
└─────────────────────────────┘
```

### 필요 이유

우리 서비스의 핵심은 단일 발언이 아니라 패널/채널의 누적 검증이다. 개별 종목 카드보다 서비스 정체성을 더 잘 보여준다.

### 추천 위치

- 패널 상세 상단
- 리그 페이지
- 공유 카드
- 주간 리포트

---

## 3.2 채널 리그 카드

### 용도

채널별 리그 등급과 변화 표시.

```text
┌─────────────────────────────┐
│ 채널 리그                    │
│ 김작가 TV                    │
│                             │
│        Major League          │
│        ▲ 승급 후보           │
│                             │
│ Score 78.4 / 100             │
│ 적중 42% · 표본 31건          │
│ 신뢰도  높음                 │
│                             │
│ 다음 평가까지 6일             │
└─────────────────────────────┘
```

### 필요 이유

사용자가 “누가 잘하나?”를 가장 직관적으로 이해하는 구조가 리그다. 이 카드는 서비스의 차별점을 직접 보여준다.

### 추천 위치

- 메인 League 섹션
- 관리자 운영실
- 주간 리포트
- 소셜 공유 카드

---

## 3.3 종목별 누가 먼저 말했나 카드

### 용도

특정 종목을 두고 여러 채널/패널이 언제 언급했는지 보여준다.

```text
┌─────────────────────────────┐
│ 삼성전자 언급 타임라인        │
│                             │
│ 03.12  김작가 TV      68,000 │
│       ━━━━━━━━━              │
│ 03.28  신사임당       71,200 │
│          ━━━━━━━━            │
│ 04.15  ○○패널        74,500 │
│              ━━━━━           │
│                             │
│ 현재 76,300원 · 05.05 기준    │
└─────────────────────────────┘
```

### 필요 이유

바이럴성이 강하다. 사용자가 궁금해하는 것은 단순히 “올랐나?”가 아니라 누가 먼저 말했는지, 누가 고점/저점에서 언급했는지, 여러 채널이 같은 종목을 동시에 말했는지다.

### 추천 위치

- 종목 상세
- 공유 리포트
- X/블로그/인스타 카드
- 이번 주 많이 언급된 종목 섹션

---

## 3.4 발언 근거 카드

### 용도

성과 숫자 뒤에 실제 근거 발언을 보여준다.

```text
┌─────────────────────────────┐
│ 근거 발언                    │
│                             │
│ “현금 비중을 줄이고           │
│  AI 인프라 쪽을 봐야 한다”    │
│                             │
│ 영상 12:34 구간               │
│ 신사임당 · 2026.04.27         │
│                             │
│ 추출 신뢰도 86%               │
│ 사람 검수 완료                │
└─────────────────────────────┘
```

### 필요 이유

성과 카드만 있으면 “이거 진짜 추천한 것인가?”라는 의심이 생긴다. 근거 발언 카드는 서비스 신뢰도를 올린다.

### 추천 위치

- 상세 페이지
- 관리자 검수 큐
- 공유 카드 하단
- 논란 방지용 evidence module

---

## 3.5 리스크/주의 카드

### 용도

성과가 나쁘거나 해석이 애매한 경우를 중립적으로 표시한다.

```text
┌─────────────────────────────┐
│ 해석 주의                    │
│                             │
│ 이 발언은 명확한 종목 추천이  │
│ 아니라 산업 전망에 가까워요. │
│                             │
│ 분류 상태    검수 필요        │
│ 신뢰도       61%              │
│ 영향도       중간             │
│                             │
│ 공개 발언 기준 후보 데이터     │
└─────────────────────────────┘
```

### 필요 이유

서비스가 커질수록 가장 위험한 것은 AI가 추천으로 오판하는 것이다. 이 카드는 신뢰와 규제 리스크를 줄인다.

### 추천 위치

- 관리자 검수 큐
- 상세 페이지의 애매한 발언
- Caution league
- 알고리즘 운영실

---

## 3.6 주간 리포트 카드

### 용도

매주 자동으로 공유 가능한 요약 카드.

```text
┌─────────────────────────────┐
│ 이번 주 투자 콘텐츠 리포트     │
│ 2026.04.29 - 05.05           │
│                             │
│ 많이 언급된 종목              │
│ 1. 삼성전자      12회         │
│ 2. SK하이닉스     9회         │
│ 3. 한미반도체     7회         │
│                             │
│ 성과 GOOD 발언     14건       │
│ 검수 필요 후보      6건       │
│ 신규 채널 후보      3개       │
└─────────────────────────────┘
```

### 필요 이유

비용 0 바이럴의 핵심 카드다. 서비스가 매주 알아서 공유할 소재를 만들어야 한다.

### 추천 위치

- 메인 주간 리포트
- 블로그 자동 포스트
- 링크드인/X 요약 이미지
- 카카오/텔레그램 공유 카드

## 3.7 승급/강등 뉴스 카드

### 용도

리그 시스템의 변화 이벤트를 뉴스처럼 보여준다.

```text
┌─────────────────────────────┐
│ 리그 변동                    │
│                             │
│ 신사임당 샘플                │
│ Minor → Major                │
│                             │
│ 승급 사유                    │
│ 최근 2회 평가 연속 기준 충족  │
│ GOOD 4 · MISS 0              │
│ 평균 +5.1%                   │
│                             │
│ Algorithm v0.1 기준           │
└─────────────────────────────┘
```

### 필요 이유

리그 시스템은 변동이 있어야 살아있어 보인다. 승급/강등 뉴스 카드는 사용자 재방문과 공유를 유도한다.

### 추천 위치

- League Board
- 주간 리포트
- 알림/이메일
- 소셜 카드

## 3.8 알고리즘 설명 카드

### 용도

점수 산정 기준을 투명하게 보여준다.

```text
┌─────────────────────────────┐
│ 점수 계산 기준                │
│ league-ranking-v0.1          │
│                             │
│ Performance   35%  ▰▰▰▰     │
│ Reliability   20%  ▰▰       │
│ Clarity       15%  ▰▱       │
│ Sample        10%  ▰         │
│ Recency       10%  ▰         │
│ Risk          10%  ▰         │
│                             │
│ 모든 점수는 공개 발언 기준     │
└─────────────────────────────┘
```

### 필요 이유

점수/리그를 쓰기 때문에 “왜 이 점수인가?”라는 질문이 반드시 나온다. 설명 가능성이 있어야 신뢰가 생긴다.

### 추천 위치

- 패널 상세
- 리그 상세
- 관리자 알고리즘 운영실
- 서비스 소개/FAQ

## 4. 화면별 추천 카드 배치

## 4.1 메인 페이지

```text
[수익률 헤드라인 카드]
이번 주 대표 검증 사례

[패널 성적표 카드 2~3개]
최근 성과 좋은 패널

[리스트 친화형 카드 여러 개]
최근 공개 발언 성과
```

## 5. 적용 우선순위

## 5.1 1차 적용: 메인/모바일 핵심

먼저 적용할 카드:

1. 수익률 헤드라인 카드
2. 리스트 친화형 카드
3. 패널 성적표 카드
4. 채널 리그 카드

이 네 개만 있어도 서비스가 훨씬 선명해진다.

## 5.2 2차 적용: 상세/신뢰 강화

그다음 적용할 카드:

1. 좌우 대비 카드
2. 발언 근거 카드
3. 리스크/주의 카드
4. 알고리즘 설명 카드

이 단계에서 서비스가 단순 랭킹이 아니라 검증 인프라처럼 보이기 시작한다.

## 5.3 3차 적용: 바이럴/소셜 확산

마지막으로 적용할 카드:

1. 종목별 누가 먼저 말했나 카드
2. 주간 리포트 카드
3. 승급/강등 뉴스 카드
4. 공유 전용 이미지 카드

이 단계가 비용 0 바이럴 엔진으로 이어진다.

## 6. 최종 추천

기본 모바일 카드:

- 대안 4 리스트 친화형
- 가장 많이 반복될 카드라서 기본값으로 적합하다.

대표 성과 카드:

- 대안 1 수익률 헤드라인형
- 메인 상단과 공유 카드에 적합하다.

상세 복기 카드:

- 대안 3 좌우 대비형
- “추천일 vs 현재” 검증 메시지를 가장 잘 전달한다.

고급 차트 카드:

- 대안 2 스파크라인형
- 실제 가격 데이터 연결 후 상세 화면에 적용한다.

추가 필수 카드:

- 패널 성적표 카드
- 채널 리그 카드
- 발언 근거 카드
- 주간 리포트 카드
- 종목별 누가 먼저 말했나 카드

한 줄 정리:

> 메인에서는 대안 1로 시선을 잡고, 대안 4로 빠르게 스캔시키고, 상세에서는 대안 3으로 검증 신뢰를 만들고, 바이럴에서는 주간 리포트/성적표/누가 먼저 말했나 카드로 공유를 유도하는 구조가 가장 좋다.

## Sketch: Panel Performance ASCII Cards

### 목적
패널/채널별 공개 발언 성과 카드를 ASCII 감성의 모바일 카드 UI로 변환했을 때 어떤 정보 위계가 좋은지 비교한다.

### 비교안
1. 수익률 헤드라인형: 수익률을 최상단 hero로 강조.
2. 미니 스파크라인 포함형: 가격 추세를 시각적으로 포함.
3. 좌우 대비형: 추천일 vs 현재를 Before/After로 분리.
4. 리스트 친화형: 여러 패널/종목을 빠르게 스캔.

### 현재 판단
- PMF 메인 리스트/패널별 성과 비교: 대안 4 기반.
- 종목 상세/공유 카드: 대안 1 기반.
- 교육/복기 콘텐츠: 대안 3 기반.
- 실제 차트 데이터가 붙으면 대안 2를 상세 카드로 검토.
